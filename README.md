# Passos Mágicos - Análise de Clusters com RAG

## 📋 Visão Geral
Sistema de análise de alunos da Associação Passos Mágicos usando 
clusterização (UMAP + HDBSCAN) e RAG para apoio pedagógico.

## 🎯 Problema Resolvido
Identificar perfis de alunos para intervenção personalizada:
- 🔴 Crítico: alunos com risco de abandono
- 🟠 Atenção: alunos estagnados
- 🟡 Em Desenvolvimento: alunos no caminho certo
- 🟢 Destaque: alunos de alto desempenho

## 🛠️ Stack Tecnológica
- Python 3.10
- UMAP + HDBSCAN (clusterização)
- FastAPI (API)
- ChromaDB + LangChain (RAG)
- Streamlit (Dashboard)
- Docker
